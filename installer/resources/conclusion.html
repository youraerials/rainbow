Next Steps

  1. The Setup Wizard should launch automatically. If not, open Terminal and run: rainbow config edit
  2. Choose your domain (e.g., yourname.rainbow.rocks)
  3. Enter your Cloudflare API token
  4. Select which services to enable
  5. Start your server: rainbow start

Quick Commands

rainbow status — Check all services
rainbow start — Start everything
rainbow stop — Stop everything
rainbow backup — Run a backup
